<b>Touchegg Installation:</b>
-------------------------------

Step 1:

Install deb file :

        sudo dpkg -i touchegg_2.0.16_amd64.deb
        
Step 2:

Enable Touchegg service

     sudo systemctl start touchegg
     
<br>
     
     sudo systemctl enable touchegg
     
<br>

     Reboot Your System
